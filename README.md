# Loan Default Risk Predictor

A Machine Learning web application built using Streamlit.

## Features

- Loan default prediction
- Random Forest Classifier
- Interactive UI
- Probability prediction
- Risk interpretation

## Tech Stack

- Python
- Streamlit
- Scikit-Learn
- Pandas
- NumPy

## Run

```bash
pip install -r requirements.txt
streamlit run loan_ui.py
